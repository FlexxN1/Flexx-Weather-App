# Weather-Flexx-App🌀🌪

Weater-App, es una pagina web que nos ayuda buscar el clima y distintos datos de un lugar en absoluto.

Recursos :

- [SweetAlert2.0](https://cdnjs.com/libraries/sweetalert)

- [OpenWeatherAPI](https://openweathermap.org/)
